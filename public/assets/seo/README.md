# Social-sharing image

`og-default.jpg` is the default Open Graph and X/Twitter sharing image for GENUS MALAYSIA SDN BHD.

Recommended production dimensions: 1200 x 630 pixels.

If the production domain changes, update the central `siteUrl` value in `src/config/seo.ts` and the absolute URLs in `public/robots.txt` and `public/sitemap.xml`.
